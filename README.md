## How to use the model?

Clone the project

```bash
  git clone https://github.com/Bot-Ro-Bot/Transfer-Learning-Brain-Tumor-Dataset-.git
```

Go to the project directory

```bash
  cd Transfer-Learning-Brain-Tumor-Dataset-
```

Install Libraries

```bash
  pip install -r requirements.txt
```

Run

```bash
  python app.py
```
